# Ruleta Online en PHP y JavaScript

## Descripción del proyecto

Este proyecto nació como un ejercicio académico en clase, con el objetivo de **crear un juego de azar usando únicamente PHP**. En un principio, el proyecto estaba pensado para **ejecutarse desde el terminal** de Visual Studio Code, funcionando como una versión básica de ruleta.

Una vez completado el proyecto inicial en PHP, decidí **diseñar un frontend con JavaScript** para mejorar la experiencia visual y permitir interacción más amigable. La interfaz todavía está en desarrollo y puede ser mejorada en futuras versiones.

---

## Características actuales

- Ruleta funcional con PHP (backend)
- Saldo virtual para apuestas
- Historial de números generados
- Interfaz inicial con HTML, CSS y JavaScript (animaciones básicas)
- Posibilidad de girar la ruleta y mostrar resultados dinámicamente

---

## Tecnologías usadas

- PHP  
- JavaScript  
- HTML5 / CSS3  
- Canvas para animaciones de la ruleta  
- (Opcional) MySQL para guardar saldo y historial de números
